Módulo app

app.texto_a_braille(texto)[fuente]

Convierte texto normal a símbolos Braille.

Parámetros:

texto (str) – El texto a convertir

Devuelve:

El texto convertido a símbolos Braille

Tipo del valor devuelto:

str

app.index()[fuente]

Ruta principal que muestra la interfaz.

app.convertir()[fuente]

Endpoint para convertir texto a Braille. Espera un JSON con el campo “texto”.

Variables globales

app.BRAILLE_MAP

dict() -> new empty dictionary dict(mapping) -> new dictionary initialized from a mapping object’s

(key, value) pairs

dict(iterable) -> new dictionary initialized as if via:

d = {} for k, v in iterable:

d[k] = v

dict(**kwargs) -> new dictionary initialized with the name=value pairs

in the keyword argument list. For example: dict(one=1, two=2)
